 Task Manager App 

A beautiful, feature-rich task management Flutter application with Material 3 design and intuitive user interface.

 Features 

 Core Features
- **Add New Tasks** with title, description, and priority levels (High/Medium/Low)
- **View Task List** in a clean, organized interface
- **Mark Tasks as Completed** with visual feedback
- **Delete Tasks** with confirmation dialog
- **Priority Management** with color-coded badges

 UI/UX Features
- **Material 3 Design** with modern aesthetics
- **Responsive Layout** works on all screen sizes
- **Visual Priority Indicators** ( High,  Medium,  Low)
- **Task Statistics Dashboard** showing progress
- **Empty State** with helpful guidance
- **SnackBar Notifications** for user actions
